**The audio controls browsers forgot to include.**

Boost quiet videos up to 500%, fine-tune equalizer settings, set automatic volume rules per site, and route audio to different output devices — all from a single popup.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

**VOLUME CONTROL**
• Volume range from 0% to 500% per tab
• Precision slider for fine-grained control
• Five customizable preset buttons for quick access
• Built-in limiter prevents clipping and protects speakers
• Badge shows current volume percentage on extension icon

**AUDIO PROCESSING**
• Balance slider for left/right panning
• Channel modes: Stereo, Mono, Swap
• Bass Boost/Cut for low frequency control
• Treble Boost/Cut for high frequency control
• Voice Boost for clearer dialogue
• Dynamic range compressor with presets: Podcast, Movie, Maximum

**VISUALIZER**
• Five styles: Bars, Waveform, Mirrored, Curve, Dots
• Click and hold to cycle through styles
• Click visualizer to jump to that tab
• Works on all sites via Tab Capture

**AUDIO MODES**
• Tab Capture (default) — works on all sites, most reliable
• Web Audio — lighter option for simple sites
• Disabled Mode — native browser volume only (0-100%)
• Per-site mode preferences saved automatically

**DEVICE SELECTION**
• Route each tab to different audio outputs
• Set a global default device for new tabs
• Useful for gaming + music, streaming + chat, work calls

**SITE RULES**
• Save settings for specific websites or domains
• Auto-apply volume, EQ, compressor, device, and more
• Syncs across devices via browser account

**PHILOSOPHY**
• No subscriptions, no premium tiers, no ads
• Full source code available on GitHub

**CONVENIENCE**
• Keyboard shortcuts: Alt+Shift+Up/Down (volume), Alt+Shift+M (mute)
• Right-click context menu with full feature access
• Play/Pause media from popup or context menu
• Mute other tabs to focus on one audio source
• Switch between audio tabs without leaving the popup
• Reset tab to defaults with one click
• Basic/Advanced mode toggle for simplified or full UI
• Built-in user guide with feature explanations

**CUSTOMIZATION**
• Light and dark themes (Morning/Bedtime)
• Drag-and-drop header layout in Settings
• Reorder or hide popup sections (Balance, Enhancements, Output, Site Rule)
• Choose between sliders or preset buttons for effects
• Configure your own volume and effect presets
• Backup and restore all settings

**ACCESSIBILITY**
• Full keyboard navigation support
• Screen reader compatible with live status announcements
